UDT: UDP-based Data Transfer Library - version 2

Yunhong Gu

 

Welcome to the UDT2 SDK documentation.

UDT is also the name of our high performance data transfer protocol - UDP-based data transfer protocol. It is designed for data intensive applications over wide area networks such as computational grids, to overcome the efficiency and fairness problems of TCP. Starting from version 2, UDT also supports user defined congestion control algorithms.

Visit http://udt.sf.net for most recent news on UDT.

Check out most current UDT release at SourceForge or from CVS.

export CVS_RSH=ssh

cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/udt login

[NOTE: when prompt for password, press the RETURN/ENTER key]

cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/udt co UDT2
