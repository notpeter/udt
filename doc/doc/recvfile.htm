UDT Reference: UDT Functions

recvfile

The recvfile method sends out a file stream.

int recvfile(

   UDTSOCKET u,
   ofstream& ofs,

   const streampos& offset,

   const streampos& size,

   const int& block = 367000
);

Parameters

u
[in] Handle to a connected socket.
ofs
[out] ofstream descriptor of the file to store the incoming data.
offset
[in] the position from where file is overwritten.
size
[in] the total size to be received.
block
[in] Optional. data block size for each receiving and writing.

Return Values

On success, the actual size of data received is returned. Otherwise, UDT_ERROR is returned and the specific error information can be retrieved using getlasterror.

Error Code Comment

5000

file system error.

5004

u is an invalid UDT socket.

6004

a non-blocking overlapped recv is in progress.

2001

connection is broken before the IO completes.

2002

u is not connected.

Description

The recvfile method reads data directly into a file stream. It is always in blocking mode. The UDT_RCVSYN option only affects recv.
